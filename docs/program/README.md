# Program Management Documentation

## English

# Program Management

This directory contains program management assets for the Nornnet project. It includes decision records, risk logs, stakeholder registers, and other artifacts essential for governing and coordinating multiple projects within the program.

### Contents

- **Decision Records**: Architectural and strategic decisions with rationales and alternatives
- **Risk Logs**: Identified risks, mitigation strategies, and current status
- **Stakeholder Registers**: Key stakeholders, their roles, and communication plans
- **Benefits Realization Plans**: Tracking and measuring program benefits and outcomes
- **Program Governance**: Policies, procedures, and oversight mechanisms
- **Resource Allocation**: Plans for budgeting, staffing, and resource distribution across projects

## Danish

# Programledelse

Denne mappe indeholder programledelsesaktiver for Nornnet-projektet. Den omfatter beslutningsregistre, risikologge, interessentregistre og andre artefakter, der er essentielle for at styre og koordinere flere projekter inden for programmet.

### Indhold

- **Beslutningsregistre**: Arkitektoniske og strategiske beslutninger med begrundelser og alternativer
- **Risikologge**: Identificerede risici, afhjælpningsstrategier og nuværende status
- **Interessentregistre**: Nøgleinteressenter, deres roller og kommunikationsplaner
- **Fordelsrealiseringsplaner**: Sporing og måling af programfordele og resultater
- **Program Governance**: Politikker, procedurer og tilsynsmekanismer
- **Ressourceallokering**: Planer for budgettering, bemanding og ressourcefordeling på tværs af projekter
