
###### 🇩🇰 Danish

# Arkitektur

Denne mappe indeholder arkitektur dokumentationen for Nornnet-projektet. Den omfatter designbeslutninger, systemdiagrammer, komponentoversigter og tekniske specifikationer, der definerer, hvordan systemet er struktureret og organiseret.

### Indhold

- **Designbeslutninger**: Vigtige arkitektoniske valg og deres begrundelse
- **Systemdesign**: Højniveaustruktur og komponenter
- **Komponentguider**: Detaljeret dokumentation for hver større komponent
- **Diagrammer**: Visuelle repræsentationer af systemarkitekturen
- **Foranalyser**: Foreløbige analyser og gennemførlighedsstudier udført før detaljeret design

###### 🇬🇧 English
# Architecture

This directory contains the architectural documentation for the Nornnet project. It includes design decisions, system diagrams, component overviews, and technical specifications that define how the system is structured and organized.

### Contents

- **Design Decisions**: Key architectural choices and their rationale
- **System Design**: High-level system structure and components
- **Component Guides**: Detailed documentation for each major component
- **Diagrams**: Visual representations of the system architecture
- **Pre-analyses**: Preliminary analyses and feasibility studies conducted before detailed design
